# tanmaya-lodhia.github.io

Personal portfolio site: research, projects, stock pitches, and blog.

Built with [Astro](https://astro.build) and Tailwind CSS. Deployed automatically to GitHub Pages on push to master.

## Development

```sh
npm install
npm run dev      # dev server at localhost:4321
npm run build    # production build to dist/
```

Blog posts are markdown files in `src/content/blog/`. All other content lives in `src/data/content.ts`.
